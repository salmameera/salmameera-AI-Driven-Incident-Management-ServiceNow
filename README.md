# AI Incident Management System – ServiceNow

This project implements an AI-powered incident management system using ServiceNow Virtual Agent and Flow Designer.

## Features
- Virtual Agent chatbot
- Issue classification
- Automated troubleshooting
- Automatic incident creation
- Assignment group routing
- Priority assignment
- Work notes automation
- Resolution notes automation

## Tools Used
- ServiceNow Virtual Agent
- Flow Designer
- Incident Management
- NLU
